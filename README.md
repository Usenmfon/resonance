# Resonance

Premium editorial music celebrity biography website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Stack

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Framer Motion

## Features

- Cinematic homepage with featured artist storytelling
- Searchable celebrity directory
- Dynamic biography pages at `/celebrities/[slug]`
- Light and dark theme toggle
- Real artist photos
- Discography, awards, stats, gallery, and legacy sections

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
